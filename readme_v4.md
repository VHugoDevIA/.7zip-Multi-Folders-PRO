# ZIP Múltiplas Pastas - PRO v4

Aplicação **PowerShell** com interface gráfica Windows Forms para automatizar a compressão de múltiplas pastas usando o motor do 7-Zip (7z.exe).  
Melhorias na interface e opções de compressão.

---

## Funcionalidades principais

- Interface gráfica (WinForms) para seleção e gestão de várias pastas numa só operação.  
- Suporte para formatos `zip` e `7z`, com nível de compressão e método configuráveis.  
- Controlo do número de threads de CPU.  
- Geração automática de nomes de arquivo únicos se já existir.  
- Possibilidade de arrastar e largar pastas para a lista.  
- Log detalhado na interface.  
- Barra de progresso e cancelamento.  

---

## Requisitos

- Windows com suporte a PowerShell.  
- 7-Zip instalado em `C:\Program Files\7-Zip\7z.exe`.  

---

## Instalação e Utilização

Semelhante às versões anteriores, com opções adicionais para threads e métodos de compressão.
