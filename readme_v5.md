# ZIP Múltiplas Pastas - PRO v5

Aplicação **PowerShell** com interface gráfica Windows Forms para automatizar a compressão de múltiplas pastas usando o motor do 7-Zip (7z.exe).  
Adiciona suporte a volumes divididos e encriptação básica.

---

## Funcionalidades principais

- Suporte a volumes divididos (multi-part).  
- Encriptação com palavra-passe (ZipCrypto).  
- Todas as funcionalidades das versões anteriores.  

---

## Requisitos

- Windows com suporte a PowerShell.  
- 7-Zip instalado.  

---

## Instalação e Utilização

Configure palavra-passe e tamanho de volumes se desejado.
