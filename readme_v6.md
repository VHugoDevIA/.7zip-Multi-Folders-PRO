# ZIP Múltiplas Pastas - PRO v6

Aplicação **PowerShell** com interface gráfica Windows Forms para automatizar a compressão de múltiplas pastas usando o motor do 7-Zip (7z.exe).  
Versão base para as melhorias subsequentes, com persistência de configurações.

---

## Funcionalidades principais

- Persistência de estado entre sessões (guarda em JSON no APPDATA).  
- Melhor gestão de cancelamento e limpeza.  
- Todas as funcionalidades das versões anteriores.  

---

## Requisitos

- Windows com PowerShell 5.1 ou superior.  
- 7-Zip instalado.  

---

## Instalação e Utilização

As configurações são guardadas automaticamente.
